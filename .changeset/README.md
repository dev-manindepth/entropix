# Changesets

Hello and welcome! This folder has been automatically generated by `@changesets/cli`, a build tool that works
with multi-package repos, or single-package repos to help you version and publish your code. You can
find the full documentation for it [in the readme](https://github.com/changesets/changesets/blob/main/README.md)

## How to add a changeset

1. Run `pnpm changeset` to create a new changeset
2. Select the packages you've changed
3. Choose the bump type (patch/minor/major)
4. Write a summary of the change
5. Commit the generated changeset file

When ready to release, run `pnpm version-packages` to consume changesets and bump versions,
then `pnpm release` to publish to npm.
